# LittleLearners Hub - E-Learning Platform

> [!IMPORTANT]
> **Status:** Phase 1 - Early Development (Live Production)
> This project is currently under active production. Please note that some dashboards are still in development and may not function as expected. Additionally, some subjects and educational materials are still being pushed to the system. Development will continue in the upcoming phases.

## 🚀 Overview
LittleLearners Hub is a modular E-Learning platform designed specifically for primary education. The system provides a tailored experience for different grade levels, ensuring that content and interface are age-appropriate.

## 🛠️ Tech Stack
- **Backend:** PHP 8.x
- **Database:** MySQL
- **Frontend:** HTML5, Vanilla CSS, JavaScript (ES6+)
- **Icons:** FontAwesome
- **Environment:** XAMPP / Apache

## 📖 How the System Works

### For Teachers
- **Dashboard:** Centralized management for all grade levels.
- **Content Upload:** Dedicated interface for uploading educational videos and creating lessons.
- **Attendance:** Track student presence daily.
- **Assessments:** Create quizzes and activities for specific lessons.

### For Students
- **Grade-Specific Portals:** Each student is directed to a specialized dashboard based on their grade level.
- **Learning Path:** Watch videos, read lessons, and complete activities.
- **Gamification:** Earn points by completing activities and watching educational content.

## 🏫 Grade Levels
The system is structured to support the following grades, each with its own modular environment:
- **Kindergarten (K):** Simplified interface with heavy focus on visual learning.
- **Grades 1 to 6:** Progressively advanced UI and content complexity as students move up.

## 📂 Project Structure
- `/admin` & `/teacher`: Administrative and content management tools.
- `/student`: Grade-specific subdirectories (K, 1, 2, 3, 4, 5, 6).
- `/api`: Backend logic for uploads, lessons, and user management.
- `/assets`: Global CSS and JS resources.
- `/uploads`: Storage for educational media.
